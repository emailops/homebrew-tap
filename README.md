# EmailOps Homebrew Tap

Official Homebrew tap for [EmailOps](https://github.com/emailops/emailops) — a privacy-first, AI-native email client for macOS.

## Install

```bash
brew install --cask emailops/tap/emailops
```

## Upgrade

```bash
brew upgrade --cask emailops
```

## Uninstall

```bash
brew uninstall --cask emailops        # remove the app
brew uninstall --cask --zap emailops  # also remove local data (mail DB, models, caches)
```

Note: `--zap` cannot remove OAuth tokens from the macOS keychain; delete those in Keychain Access if needed.

The cask in this tap is generated by `scripts/generate_cask.sh` in the main repo — see `homebrew/README.md` there for the release flow.
